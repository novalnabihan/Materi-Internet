Apa itu Browser?

Apa itu Browser?

Di zaman serba teknologi seperti ini, semakin banyak orang yang sudah menggunakan internet di dalam kehidupan sehari-harinya, termasuk dalam melakukan sebuah pekerjaan.

Ketika mendengar istilah browser otomatis kamu juga terfikirikan internet, kan? Yup. Tapi, apa itu web browser?

Pengertian browser adalah salah satu jenis perangkat lunak atau software yang pada umumnya di gunakan dan dimanfaatkan untuk membuka suatu halaman di situs internet. Browser juga di sebut dengan software yang berfungsi sebagai penerima, pengakses, dan penyaji berbagai informasi di internet.

Di dalam web browser terdapat konten berupa gambar, audio, video, dan halaman web. Konten-konten tersebut di hubungkan menggunakan hyperlink dan diklasifikasikan dengan URL (Uniform Resources Identifiers).

Web browser pertama hadir dengan nama WorldWideWeb dan di temukan oleh Tim Berners-Lee pada tahun 1990, yang kemudian di ubah menjadi Nexus. Namun, browser grafis pertama dan yang banyak di gunakan untuk membantu popularitas naik ke Internet adalah NCSA Mosaic.

Dengan web browser lah pengguna internet dapat berselancar dan mencari apa saja informasi atau data yang di perlukan. Browser menjadi dasar setiap pencarian informasi di seluruh penjuru dunia sebagai tempat yang menampung serta menampilkan semua yang kamu minta pada kolom pencarian.

Lalu apa bedanya browser dengan search engine? Singkatnya seperti ini!

  • Browser adalah software yang terinstall pada perangkat komputer, sedangkan search engine adalah program yang terdapat di internet.
  • Browser adalah software yang berguna untuk membuka halaman website dengan format www, sedangkan search engine berguna untuk menampilkan daftar website yang memuat kata yang di cari.
  • Kamu tidak bisa membuka halaman search engine tanpa browser. Search engine merupakan bagian dari objek internet yang bisa di akses menggunakan sebuah browser.
  • Sekarang browser sudah terintegrasi dengan search engine masing-masing, sehingga akan sedikit sulit membedakan antara keduanya.

Fungsi Web Browser

Setelah sedikit paham mengenai apa itu web browser, yuk kita bahas tentang kegunaan dan fungsi web browser itu sendiri. Fungsi web browser adalah:

Mempermudah dalam mencari informasi di internet

Ketika kamu ingin browsing atau berselancar di internet, pasti akan membutuhkan web browser untuk medianya. Benar bukan, Sob? Nah, web browser adalah software yang fungsinya mengarahkan user menuju alamat website yang kamu minta.

Menyimpan data-data di internet

Saat menggunakan browser, pasti kamu pernah mendengar bookmark? Bookmark ini sendiri adalah fitur dari browser yang berfungsi sebagai penyimpanan data-data di internet yang suatu saat bisa kamu buka lagi tanpa perlu mengunjungi search engine.

Membuka file dengan ekstensi tertentu

Selanjutnya, fungsi web browser adalah sebagai media untuk membuka file dengan ekstensi tertentu. Misalnya, ketika kamu membuka pdf dengan browser. Atau bagi kamu yang suka otak-atik coding, pasti kamu akan melihat hasilnya melalui web browser.

Mendukung pemakai search engine

Seperti yang sudah di bahas tadi, browser dan search engine tidak bisa di pisahkan. Sebab search engine adalah bagian dari web yang bisa memudahkan pengguna untuk mencari data maupun informasi di internet.

Menjaga keamanan website

Apakah kamu tahu siapa yang menjalankan proses verifikasi atau authentication? Yup, benar sekali, web browser! Jadi, dapat di katakan bahwa fungsi web browser adalah memastikan halaman website itu aman atau berbahaya untuk di gunakan. Sebab saat ini banyak website sudah terkena malware maupun di pasang software khusus untuk mencuri data pada internet. Jadi, hati-hati ya Sob!

Cara Kerja Browser

Web browser adalah software yang bisa membantumu menemukan website yang ingin kamu kunjungi dengan cara berikut:

  1. User sedang mengakses website dengan mengetikkan alamat pada situs maupun URL di address bar. Misal youtube.com.
  2. Lalu, web browser menerima request tersebut dan akan melakukan fetching atau pengambilan data pada DNS server.
  3. Data yang sudah di ambil tadi berupa IP dari request yang diketikkan user (www.youtube.com). Web browser telah mendapat IP dari www.youtube.com.
  4. Web browser mengakses server dengan IP yang sudah didapatkan dari DNS server.
  5. Server memberikan data konten dari www.youtube.com dalam bentuk HTML dan file lain seperti PHP dan CSS.
  6. Terakhir, web browser akan menampilkan konten yang sesuai dengan request user.

Kira-kira seperti itulah cara kerja web browser dalam melaksanakan tugasnya sebagai media untuk mencari informasi di internet.

Macam-Macam Web Browser

Ada banyak macam-macam web browser sekarang ini. Namun, tentunya mereka memiliki kelebihan dan kekurangannya sendiri. Itulah mengapa beda orang beda juga web browsernya! Tergantung dengan kenyamanan dari web browser itu sendiri. Berikut macam-macam browser yang sering di gunakan:

Google Chrome

Hampir 60% pengguna internet di dunia menggunakan Google Chrome sebagai browser. Browser dari Google ini hampir di gunakan di seluruh dunia khususnya pada negara-negara di benua Eropa, Asia, dan Amerika.

Mozilla Firefox

Performa dari Mozilla ini tidak perlu di ragukan lagi, sebab contoh browser ini memiliki tingkat pengamanan super ampuh di banding yang lain. Sekitar 1 dari 6 jumlah pengguna internet di dunia menggunakan Mozilla sebagai web browsernya.

Opera Mini

Saat ini Opera juga merupakan salah satu dari macam-macam web browser yang sedang populer. Setidaknya 6% dari pengguna internet di seluruh dunia menggunakannya untuk perangkat komputer, dan 11% dari pengguna internet menggunakannya untuk perangkat mobile.

Internet Explorer

Meski pengguna (user) internet explorer semakin sedikit, tetap saja browser ini masih ada yang menggunakannya. Pada tahun 2002-2003, Internet Explorer pernah di gunakan sampai 95% pengguna internet.

UC Browser

Kelebihan dari UC browser ini adalah bisa mengecilkan data browsing, sehingga bisa semakin hemat data. Inilah yang menjadikan UC Browser menjadi peramban favorit mereka! UC Browser adalah contoh web browser yang mampu mengelola pengunduhan dengan baik. Namun di balik itu, desainnya kurang menarik jadi pengguna akan sedikit bosan dengan desain yang itu-itu saja.

Safari

Hampir 16% pengguna internet di dunia menggunakan Safari. Aplikasi buatan Apple Inc ini di rilis pada tahun 2003 untuk perangkat komputer dan 2007 untuk perangkat mobilenya.

Samsung Internet Browser

Ini adalah salah satu contoh web browser yang bisa di pakai di android. Memang awalnya browser ini hanya dapat di gunakan untuk Samsung saja, namun sekarang web browser ini bisa di pakai di android apa saja sebab bisa di unduh di playstore.

Browser ini memiliki kelebihan, bebas dari iklan dan berkecepatan tinggi. Sedangkan kekurangannya adalah tampilannya yang kurang menarik jika di banding dengan web lain.

Microsoft Edge

Microsoft Edge adalah browser yang cepat dan aman dari Windows. Terdapat beberapa alasan juga mengapa kamu harus memilih Microsoft Edge sebagai browser, seperti fitur reading list, reading mode, peningkatan keamanan, cepat, tidak berantakan, customizable start page, coding support, dan juga extensions.

Netscape Navigator

Yang dimaksud Netscape navigator browser adalah web browser yang sudah lebih dulu populer sebelum lainnya, tepatnya pada era 90-an. Netscape Navigator juga merupakan salah satu browser tertua dengan maraknya evolusi internet.

Flock

Flock adalah contoh web browser spesialis penyedia jejaring sosial dan merupakan browser sosial internasional dunia pertama yang berbasis pada firefox. Banyak fitur yang menarik, dan tentu bisa memberikan pengalaman berkomunitas sosial di internet sambil browsing.

Kesimpulan

Bagaimana? Meskipun kamu sudah sering menggunakan browser, tapi informasi di atas tentu menambah insight baru bukan?

Dilihat-lihat, memang web browser adalah aplikasi yang sering sekali kita gunakan, baik untuk bekerja, mengerjakan tugas sekolah, hingga bermain games. Bayangkan jika web browser tidak ada, bagaimana caranya kita bisa mengakses search engine?
